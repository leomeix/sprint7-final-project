Objetivo 
El objetivo de este análisis es determinar el comportamiento de los clientes por segmentos de edad y de uso para poder ver donde existe una mayor cantidad y donde podemos reforzar esfuerzos

Datasets utilizad
Se utilizaron 3 datasets para este análisis:
plans que contiene información básica de los planes
users que contiene información de los usuarios como nombre, id, edad y ciudad
usage que contiene el uso de cada id de usuario

Etapas del analisis
1 - carga y exploración 
2 - identificación de problemas con la data
3 - limpieza básica
4 - Revisión de medidas clave
5 - Creación de visualizaciones
6 - Segmentación
7 - Insigth ejecutivo

Como ejecutar el archivo
El archivo S7 Version-Estudiante-Project-ConnectaTel.ipynb se encuentra cargado directamente en esta plataforma por lo que se puede visualizar sin problema

Guia de reproducción
Para poder reproducir este analisis se deben seguir en primer lugar las estapas descritas anteriormente y para más detalle se pueden seguir las indicaciones en el analisis marcados con "#"
A nivel general se puede resumir asi:
Primero se deben cargar los y los dataset, posteriormente se explora la información para revisar si existen datos nulos, faltantes o con formato diferente.
En la limpieza se corrigen estos hallazgos dependiendo el caso y se cambian los formatos necesarios.
Posterior se calculan las metricas claves que nos ayudan a ver las tendencias de la información.
Para complementar esto se creas histogramas y boxplots que nos ayuden a confirmar los números calculados anteriormente.
Luego se hace la segmentación creando las columnas necesarias siguiendo las reglas que crean cada nuevo grupo.
Con esta información se hacen los ultimos analisis y graficos que nos ayudan a contestar las pregunta clave y con esto creamos nuestro insigths ejecutivos.
